HOW TO INSTALL VoiceFlow AI Chrome Extension
=============================================

STEP 1: Add icon images (required)
- Create 3 PNG images named: icon16.png, icon48.png, icon128.png
- Put them in a folder called "icons" inside this chrome-extension folder
- Use your VoiceFlow AI logo (purple mic icon)
- Sizes: 16x16px, 48x48px, 128x128px
- OR: Remove the "icons" section from manifest.json and Chrome will use a default icon

STEP 2: Open Chrome Extensions page
- Open Chrome browser
- Type in address bar: chrome://extensions
- Press Enter

STEP 3: Enable Developer Mode
- Look for "Developer mode" toggle in top-right corner
- Turn it ON

STEP 4: Load the extension
- Click "Load unpacked" button (appears after Developer mode is on)
- Select this "chrome-extension" folder
- Click "Select Folder"

STEP 5: Done!
- VoiceFlow AI icon appears in your Chrome toolbar
- A floating mic button appears on every website
- Click it to start recording in any language
- Text is inserted directly at your cursor

SHORTCUT: Press S twice quickly (SS) on any page to toggle recording

WORKS ON: Facebook, Gmail, WhatsApp Web, Google Docs, Twitter, LinkedIn, anywhere!
